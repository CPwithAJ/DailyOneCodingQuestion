# Day 11 Question

Given an integer value, convert it into roman number.

---

### Problem Explanation

Roman numbers are represented by below characters:

| Number | Roman |
|:---:|:---:|
| 1 | I |
| 5 | V |
| 10 | X |
| 50 | L |
| 100 | C |
| 500 | D |
| 1000 | M |

Some other special cases

| Number | Roman |
|:---:|:---:|
| 4 | IV |
| 9 | IX |
| 40 | XL |
| 90 | XC |
| 400 | CD |
| CM | 900 |

---

#### Example

Input: `3`
Output: `III`

Input: `58`
Output: `LVIII`
