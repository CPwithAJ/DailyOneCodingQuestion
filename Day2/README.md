# Day 2 Question

Given two non empty Linked List with non negative numbers, and numbers are stored in reverse order having single digit. Add the two list and return the result as a linked list.

---

#### Example
Input: `[ 3 -> 4 -> 5 ] + [ 1 -> 2-> 3]`

Output: `[ 8 -> 6 -> 4 ]`

---

#### Explanation
`5 -> 4 -> 3` **+** `3 -> 2 -> 1` **=** `8 -> 6 -> 4`
