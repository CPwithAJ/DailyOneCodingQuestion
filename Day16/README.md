# Question 16

Given an array of non repeating numbers and a key, find all the unique combinations in that array, where the sum of those combination is equal to the key.

#### Example 1:
**Input:**
    `Array = [2,3,6,7]`
    `key = 7`,

**Output:**
```
    [
        [7],
        [2,2,3]
    ]
```

#### Example 2:

**Input:** `Array = [2,3,5]`, `key = 8`

**Output:**
```
[
  [2,2,2,2],
  [2,3,3],
  [3,5]
]
```
